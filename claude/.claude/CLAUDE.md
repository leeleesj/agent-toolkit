# 글로벌 개인 설정

## 언어
- 모든 답변은 한국어로 작성
- 코드 주석도 한국어로 작성

## 커밋 규칙
- Conventional Commit 형식만 사용: type(scope): 설명
- type: feat, fix, refactor, docs, chore, test, perf
- 제목 50자 이내, 한국어, 현재형

## 코드 스타일
- 함수/변수명은 snake_case (Python), camelCase (JS/TS)
- 함수는 단일 책임 원칙 준수
- 하드코딩 금지 (API 키, URL, 매직 넘버)

## 답변 스타일
- 결론 먼저, 설명은 그 다음
- 불필요한 반복 설명 생략
- 코드 예시는 실행 가능한 수준으로

## 학습 스타일
- 코드 설명 시 해당 논문/수식과 연결해서 설명
- 개념 설명 시 직접 구현 예시 포함
- 내 이해가 틀렸을 때 왜 틀렸는지 명확히 짚어주기
