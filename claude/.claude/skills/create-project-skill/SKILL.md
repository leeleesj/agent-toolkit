## 절차
1. 스킬명과 목적 파악
   - 인자로 받으면 바로 진행
   - 없으면 질문: "어떤 스킬을 만들까요? 스킬명과 목적을 알려주세요"
2. references/skill-format.md 읽고 포맷 기준 숙지
3. 관련 코드/파일 탐색
   - 스킬 목적과 관련된 파일 읽기
   - 반복되는 절차, 주의사항, 규칙 파악
   - 관련 코드 없으면 목적 기반으로 초안 작성
4. SKILL.md 초안 작성
5. 초안 보여주고 논의 후 수정
6. 확인되면 .claude/skills/스킬명/SKILL.md 저장

## 행동 규칙
- 저장 위치: .claude/skills/스킬명/SKILL.md
- 이미 존재하면 덮어쓰기 전에 반드시 확인
- 직접 수정은 "수정해줘" 요청 시에만 실행
- references/ 생성은 필요할 것 같을 때만 제안

## references/ 제안 기준
- 참조할 문서/가이드가 이미 존재할 때
- 절차가 복잡해서 상세 문서가 필요할 것 같을 때
- 그 외에는 SKILL.md만 생성
